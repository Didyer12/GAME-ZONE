Descargar PCSX2 Emulador de PS2 para PC es un potente emulador para la consola de sobremesa Sony PlayStation 2 con el que podremos reproducir todos los juegos de la popular máquina en nuestro PC con el aliciente de poder configurar el apartado visual para que luzca incluso mejor que el de la consola original.

Para configurarlo correctamente deberás tener el archivo de la BIOS de alguna región de PlayStation 2 no incluido en el programa. Puedes usar algún buscador como Google para encontrar los archivos necesarios.

Emulador para PC NDS; Es el primer emulador de Nintendo DS, está programado en C++ y su código fuente fue publicado bajo licencia GPL. Actualmente se encuentra en fase de desarrollo y por el momento sólo puede ejecutar homebrew. Citra también está empezando a ejecutar algunos juegos comerciales, pero aún presentan muchos fallos y por lo tanto no se juegan de manera aceptable, aunque a pesar de ello, ha podido emular varios juegos a una velocidad aceptable debido al render por software, en lugar de por hardware.

Citra es un emulador de Nintendo DS para PC capaz de correr juegos comerciales al 100% de su velocidad, además de incorporar una serie de características para mejorar su representación visual más allá de la humilde resolución que utiliza portatil de Nintendo.

VisualBoyAdvance es un emulador libre distribuido bajo los términos de la GNU General Public License. Este programa emula software creado para la Game Boy, Super Game Boy, Game Boy Color y Game Boy Advance; todas ellas consolas portátiles creadas por Nintendo. Desde 2006, VisualBoyAdvance es uno de los emuladores más populares para Windows, aunque también es compatible con multitud de sistemas operativos como Linux, BSD, MAC OS X o BeOS. También ha sido portado para la consola Wii.

 

 

 

 

 

 

 

 

µTorrent (también conocido como uTorrent, cuya pronunciación correcta sería muTorrent, y abreviado como µT o uT) es un cliente BitTorrent freeware desarrollado por BitTorrent, Inc.. Está disponible para los sistemas operativos Microsoft Windows y Mac OS X. También está planeada una versión nativa para GNU/Linux.​ Todas las versiones de µTorrent están escritas en C++ y cuenta con traducciones a 54 idiomas.​ Está diseñado para usar pocos recursos del ordenador y ofrecer las funcionalidades de otros clientes como Vuze o BitComet.
